# Password-Generator
Этот генератор позволяет самостоятельно выбрать длину пароля. Вы можете создать короткий ключ для менее важных задач или длинный и сложный для максимальной защиты. Инструмент использует буквы в разном регистре, цифры и специальные символы для генерации стойких комбинаций. Просто укажите нужное количество символов для получения надёжного результата.

This generator allows you to choose your password length. You can create a short key for less critical tasks or a long, complex one for maximum protection. The tool uses uppercase and lowercase letters, numbers, and special symbols to generate strong combinations. Just specify the desired character count for a secure result.
